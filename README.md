# Kendi-Yapay-Zeka-Ajan-n-z-Se-in
Bu repository, kendi yapay zeka ajanınızı oluştururken gerekli bilgileri bulmanız ve bir yol gösterici olması için hazırlanmıştır. 


Bu depo, siber güvenlik operasyonlarında "Sıfır Güven" (Zero-Trust) mimarisini temel alarak oluşturulmuş **yerel yapay zeka ajanlarının** yapılandırma dosyalarını (Modelfile) ve altyapı notlarını içerir. 

Bulut API'lerine veri sızdırmadan (Data Exfiltration), %100 operasyonel gizlilik (OpSec) ile kendi bilgisayarınızda çalıştırabileceğiniz güvenlik ajanları ağıdır.
